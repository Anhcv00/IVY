


- Các việc cần làm
	- Xây dựng trang admin website
		+ Menu: 
			- Tổng quan. ( thanh nav thể hiện)
			- Quản lí đơn hàng.( in ra danh sách đơn hàng.)
			- Quản lí danh mục.
			- Quản lí sản phẩm.
			- Quản lí tài khoản.
				+ Thay đổi thông tin
				+ Thoát.
		+ nav: 
			- Tổng số đơn hàng.
			- Số đơn hàng mới.
			- Số đơn đã chưa xác nhận.		
			- Số đơn hàng hủy.
			- Doanh thu hôm nay.
			- Doanh thu tuần này.
			- Tổng doanh thu.	
		
	- Thiết kế cơ sở dữ liệu trang web
		+ Xây dựng bảng quản trị website (admin)
		+ Xây dựng bảng danh mục sản phẩm (catalog)
		+ Xây dựng bảng sản phẩm (product)
		+ Xây dựng bảng giao dịch (transaction)
		+ Xây dựng bảng đơn hàng (order)
	- Phân tích các chức năng của website
		+ Nhân viên :
			- Đăng nhập
			- tìm kiếm
			- Xem, sửa tất cả các sản phẩm.
			- Xem tất cả đơn hàng
			- Duyệt đơn hàng đang chờ.
		+ Admin: Giống với nhân viên nhưng có thêm các tính năng.
			- Xem, thêm, sửa, xóa thông tin nhân viên.
			- Xóa tất cả các sản phẩm.
			- Thay đổi tình trạng đơn hàng.

	- Danh mục sản phẩm: stt, id ,Danh mục, Tùy biến ( sửa, xóa)
	- Danh sách sản phẩm: STT, id. tên sản phẩm
	- Sản phẩm nam: STT, mã, tên, ảnh, danh mục, giá, số lượng. 
	- Sản phẩm nữ : STT, mã, tên, ảnh, danh mục, giá, số lượng.
	- Sản phẩm trẻ em: STT, mã, tên, ảnh, danh mục, giá, số lượng.
